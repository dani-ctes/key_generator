## Getting Started

### Prerequisites

* Requirement
  ```sh
  pip install -r requirement.txt
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dani-ctes/key_generator.git
   ```
2. Create environment and run it
   ```sh
   python -m venv env_x
   source ./activate
   ```
3. Install requirement

   ```sh
   pip install -r requierement.txt
   ```
4. Run
   ```sh
   python key_generator.py
   ```
